# SISA
Simple Instruction Set Architecture (SISA) open assembler

```
Usage:  
       sisa [flags] infile
    or
       sisa infile [flags]  
Options:
    -h        print this summary
    -i        assemble inline instruction (-i "ADD R0, R0, R3")
    -o        specify ouput file path
    -v        increase verbosity
```
